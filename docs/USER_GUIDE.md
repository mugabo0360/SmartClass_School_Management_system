# SmartClass User Guide (Quick)

## 1. Create Account
1. Open the app home page.
2. Click **Get Started Free**.
3. Fill school and personal details.
4. Accept legal checkboxes.
5. Submit and go to **Check Inbox** page.
6. Verify email, then login.

## 2. Core Daily Workflow
1. Login as Admin/Teacher.
2. Add students and classes (via dashboard features).
3. Enter assessments.
4. Generate remarks with AI route.
5. Share progress with parents.

## 3. CRM + Messaging Workflow
1. Create contacts in backend CRM.
2. Create templates for common messages.
3. Create campaign.
4. Send campaign to selected audience.
5. Watch analytics for delivered count.

## 4. Data Protection Workflow
1. Use only official school accounts.
2. Avoid entering unnecessary personal data.
3. Review AI text before publishing reports.
4. Export data each term for backup.

## 5. Common Issues
- **No verification email:** check spam/promotions.
- **Cannot login:** reset password or verify email.
- **Campaign not sending:** ensure contacts exist first.
